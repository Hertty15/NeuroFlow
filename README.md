# 🧠 NeuroFlow

**It helps with doing work but not getting burnt out doing work**

 **Live Demo:** [https://hertty15.github.io/NeuroFlow/]

## 🎯 What is it?
I built this because I needed something doesnt just show what all work I have, it also shows work depending how much energy I have. I can also set a timer so i can take breaks.

## ✨ Features
- **Focus Timer:** The timer helps with how much time to work for.
- **Task List:** You can sort tasks depending on how much energy it will take you or how much you need.
- **Dopamine Boost:** Gives you something to do for a small break to increase your energy.

## 🛠️ How I Built It
- **Languages:** HTML, CSS, JavaScript
- **Tools:** VS Code, GitHub

##  How to Use It
1. Step 1: Select your energy level
2. Step 2: Add a task
3. Step 3: Watch it filter
4. Step 4: Set a timer
